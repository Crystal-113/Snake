# Snake
# 关于

A simple Snake game written in C++ using SDL2.
这是一个用C++和SDL2写的极简贪吃蛇游戏。

# Build
# 构建

```bash
cmake -B build
cmake --build build
```

# To Run in Ubuntu
# 在Ubuntu上运行

```bash
sudo apt install libsdl2-2.0-0 libsdl2-image-2.0-0 libsdl2-ttf-2.0-0
./build/Snake
```

# Requirements
# 要求

Windows 10/11 x64
Ubuntu 24.04 或其他安装了 SDL2、SDL2_image、SDL2_ttf 的 Linux x64 系统

# Dependencies
# 依赖

SDL2
SDL2_image
SDL2_ttf
