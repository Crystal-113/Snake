# Snake | 关于
A simple Snake game written in C++ using SDL2.
这是一个用C++和SDL2写的极简贪吃蛇游戏。

# Build | 构建
```bash
cmake -B build
cmake --build build
```
# 安装依赖
## Windows
No operations needed.
不需要任何操作。

## Ubuntu & Debian
```bash
sudo apt install libsdl2-2.0-0 libsdl2-image-2.0-0 libsdl2-ttf-2.0-0
```

# Arch Linux
```bash
sudo pacman -S sdl2 sdl2_image sdl2_ttf
```

# Requirements | 系统要求
Windows 10/11 x64
Linux x64

# Dependencies | 依赖
SDL2
SDL2_image
SDL2_ttf
